# 🏏 Cricket Player Analytics System

A Data Warehousing & Data Mining project that analyzes cricket player performance using recent match data and predicts player form and next-match performance.

## 📌 Features

* Player form classification (GOOD / AVERAGE / POOR)
* Last 10 match performance analysis
* Next match run prediction (trend-based)
* Player clustering using K-Means
* Interactive player search system

## 🧠 Techniques Used

* Decision Tree Classification
* K-Means Clustering
* Statistical Analysis
* Weighted Moving Average Prediction
* Data Preprocessing & Feature Engineering

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* IPL Ball-by-Ball Dataset

## ▶️ How to Run

1. Clone repository:

```
git clone https://github.com/chetansatvik18/Cricket-Player-Analytics.git
```

2. Install dependencies:

```
pip install pandas scikit-learn
```

3. Download IPL dataset and place inside:

```
Dataset/IPL.csv
```

4. Run:

```
python Python/player_analysis.py
```

## 📊 Example Output

* Displays last 10 match scores
* Predicts player form
* Shows expected runs next match
* Groups players by performance clusters
